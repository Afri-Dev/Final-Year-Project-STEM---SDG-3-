HOW TO RUN THE APP
==========================================

1. OPEN TERMINAL in this folder (stem_learning_app)

2. RUN ONE OF THESE COMMANDS:

   For Android:
   npm run android

   For iOS (Mac only):
   npm run ios

   For Web:
   npm run web

   Or just start the server:
   npm start
   (Then press 'a' for Android, 'i' for iOS, 'w' for Web)

==========================================
TROUBLESHOOTING
==========================================

Problem: White screen or crash
Fix: npm start -- --reset-cache

Problem: Can't find app on phone
Fix: Install "Expo Go" app from Play Store/App Store

Problem: Metro bundler stuck
Fix: Close terminal, run npm start again

Problem: TypeScript errors
Fix: Restart VS Code
